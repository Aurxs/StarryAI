# page: workbench

工作台主页面，组合画布、审查与运行入口。

## 当前实现补充

- 页面骨架改为“画布优先”的悬浮式 HUD：
  - 左上角项目下拉（当前为演示数据）；
  - 右上角单一“测试运行”入口；
  - 左下角撤回/重做与操作历史；
  - 底部中间审查栏与审查问题抽屉。
- 底部三块 HUD（撤回区/审查栏/缩放区）已统一等高，并修复抽屉打开时可能出现的重叠问题。
- 整体视觉更新为浅灰蓝系浮空卡片风格（Dify 风格方向）：弱对比边框 + 轻阴影 + 半透明白底。
- 审查栏去除外层重复边框，边框保持与其它组件一致的中性色；通过/失败状态仅通过绿/红外发光提示。
- 节点选中后会打开右侧节点编辑抽屉，并推动底部 HUD 向左偏移，避免遮挡。
- 节点选中后，右上角运行区也会同步左移，避免与右侧抽屉重叠。
- 图编辑后自动触发审查（500ms 防抖）；仅当无 `error` 时允许运行。
- 图校验链路内置同步托管重算：
  - 仅在“同步发起器 <-> 同步执行节点”关系或发起器同步参数变化时触发；
  - 命中变化后自动将发起器同步参数下发到下游同步执行节点；
  - 关系断开后自动 reset 下游托管同步参数。
- 右上角运行区新增同步指标摘要（轮询 `/runs/{id}/metrics`）：
  - 同步提交次数；
  - 同步中止次数；
  - 中止原因聚合。
- 语言切换保留（`zh-CN`/`en-US`）并持久化到 `localStorage`。
