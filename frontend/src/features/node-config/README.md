# feature: node-config

节点属性表单与参数校验 UI。

## 当前实现补充

- 节点配置面板文案与提示信息已迁移到语言包。
- 同步参数托管规则（第三批）：
  - 同步发起器可编辑：`sync_group`、`ready_timeout_ms`、`commit_lead_ms`；
  - `sync_round` 在前端统一只读展示（自动轮次）；
  - 同步执行节点的四个同步参数全部只读，并显示来源发起器 `node_id`。
- 同步执行节点仍可编辑业务运行参数：
  - 配置 JSON 编辑区仅编辑非托管字段；
  - 保存时会自动合并回当前托管同步参数，避免被手工覆盖。
- 同步字段保存时执行前端校验（必填与整数范围）并回写节点 `config`。
